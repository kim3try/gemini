캔버스 시작하기

Gemini Canvas는 몇 가지 간단한 단계만으로 시작할 수 있습니다. AI와 함께 아이디어를 현실로 만들어 보세요.

🌐

1. 접속

`gemini.google.com`에 접속하여 Gemini를 엽니다.

→
➕

2. 캔버스 선택

프롬프트 입력창 아래 '+' 버튼을 누르고 '캔버스'를 선택합니다.

→
✍️

3. 프롬프트 입력

원하는 작업 내용을 구체적으로 입력하여 초안을 만듭니다.

주요 기능 및 활용법

캔버스는 문서 작성부터 코드 개발, 콘텐츠 제작까지 다양한 작업을 지원합니다. 각 기능을 탐색해 보세요.

콘텐츠 제작

AI와 함께하는 스마트한 문서 작업

  • ✓초안 생성: 보고서, 블로그 글 등 다양한 문서의 초안을 AI가 빠르게 만들어 줍니다.
  • ✓실시간 수정: "더 간결하게", "전문적으로" 등 간단한 요청으로 문체의 톤과 스타일을 쉽게 변경할 수 있습니다.
  • ✓공동 작업: 완성된 문서를 Google Docs로 내보내 팀원들과 효율적으로 공유하고 협업할 수 있습니다.

아이디어를 현실로 만드는 코드 개발

  • ✓코드 생성: HTML, React 등 다양한 웹 코드를 AI가 신속하게 작성해 줍니다.
  • ✓실시간 미리보기: 생성된 코드가 웹에서 어떻게 보일지 즉시 확인하며 수정할 수 있어 개발 효율이 높아집니다.
  • ✓디버깅: 코드의 오류를 찾거나 더 나은 구조로 개선해 달라고 요청하여 코드 품질을 향상시킬 수 있습니다.

다채로운 콘텐츠 제작

  • ✓인포그래픽: 복잡한 보고서나 데이터를 한눈에 파악할 수 있는 시각적인 인포그래픽으로 변환합니다.
  • ✓퀴즈: 특정 주제에 대한 학습용 퀴즈를 생성하여 교육 자료로 활용할 수 있습니다.
  • ✓오디오 개요: 문서를 팟캐스트 형식의 오디오 요약본으로 만들어 이동 중에도 들을 수 있습니다. (현재 영어로만 제공)

캔버스 활용 팁

몇 가지 팁을 활용하여 Gemini Canvas를 더욱 효과적으로 사용해 보세요.

🎯

구체적으로 요청하기

원하는 결과를 얻으려면 프롬프트를 최대한 구체적이고 명확하게 작성하는 것이 좋습니다.

💾

자동 저장

캔버스에서의 모든 변경 내용은 자동으로 저장되므로, 작업 내용이 사라질 걱정이 없습니다.

🖥️

데스크톱 활용

텍스트 스타일 변경이나 서식 지정 등 더 많은 기능은 데스크톱 웹 버전에서 편리하게 이용할 수 있습니다.
